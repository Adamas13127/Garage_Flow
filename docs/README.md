# Documentation GarageFlow

Ce dossier contient la documentation fonctionnelle et technique du projet GarageFlow.

## Organisation

* `reference/` : documents sources et PDF de reference.
* `fonctionnelle/` : besoins, parcours utilisateurs et regles fonctionnelles.
* `technique/` : architecture, choix techniques et notes de conception.
* `mcd/` : modele conceptuel de donnees.
* `mld/` : modele logique de donnees.
* `api/` : documentation des endpoints et contrats API.
* `oral/` : supports et notes pour la presentation orale.

## Regle importante

Les documents places ici aident a construire le projet, mais `AGENTS.md` reste la source de verite principale pour le scope MVP et les consignes de developpement.
