# Documents de reference

Ce dossier peut contenir les PDF sources du projet GarageFlow : grille d'evaluation, cahier des charges, maquettes, MCD, MLD, dictionnaire de donnees, matrice des droits, architecture ou plan de developpement.

Ces documents servent de support de reference.
Ils ne demandent pas automatiquement de developper toutes les fonctionnalites qu'ils mentionnent.

Avant toute implementation, il faut verifier le scope MVP dans `AGENTS.md`.
En cas de contradiction entre un PDF et `AGENTS.md`, demander confirmation avant de developper.
